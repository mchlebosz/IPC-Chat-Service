void removeData ( const char *  file_name,
const char *  data 
)

It opens the file in read mode, creates a temporary file, reads the contents of the original file into a buffer, checks if the data to be removed is in the buffer, and if it is, it writes the buffer to the temporary file. If the data is not in the buffer, it does not write the buffer to the temporary file. After the file has been read, the original file is closed, the temporary file is closed, the original file is removed, and the temporary file is renamed to the original file

Parameters
file_nameThe name of the file to remove data from.
dataThe data to be removed from the file.
